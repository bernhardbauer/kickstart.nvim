# Install dependencies

## HomeBrew

```
brew install dotnet
brew install ollama
brew install pkl-lsp
brew install prettier
brew install prettierd
brew install --cask claude-code
```

### Environment variables (.zshrc)

```
export DOTNET_ROOT="/opt/homebrew/opt/dotnet/libexec"
```
